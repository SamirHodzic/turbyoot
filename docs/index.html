Turbyoot

0.2.8

A modern, simple and intuitive Node.js web framework that makes building APIs simple and enjoyable.

$ npm install turbyoot
import { Turbyoot } from 'turbyoot';
const app = new Turbyoot();
const port = 3000;

app.get('/', (ctx) => {
  ctx.ok({ message: 'Hello World!' });
});

app.listen(port, () => {
  console.log(`Turbyoot is running on port ${port}`);
});

Zero Dependencies

Lightweight framework with no external runtime dependencies, keeping your application lean and fast.

Fluent API

Chainable, intuitive syntax for clean and readable code that makes routing a pleasure.

Resource Routing

Automatic CRUD routes with custom handlers and filtering, reducing boilerplate code significantly.

Grouped Routes

Organize routes with prefixes and shared middleware for better code organization and maintainability.

Plugin System

Extend functionality with a clean plugin architecture that makes your code modular and reusable.

TypeScript Support

Built with TypeScript, providing full type safety out of the box for a better development experience.
